# MRO Aerolíneas Argentinas - División Motores

## Propósito del Sitio
Esta página web tiene como objetivo presentar las capacidades técnicas del taller MRO de Aerolíneas Argentinas, 
específicamente en el mantenimiento, reparación y recorrida general de motores de última generación.

## Motores Cubiertos
- **CFM56-7B**: Equipando la flota Boeing 737 NG.
- **CF34-10E**: Motorización de la flota Embraer 190.
- **CF6 / Trent 700**: Motores de fuselaje ancho (Airbus A330).
- **LEAP-1B**: Tecnología de punta para el Boeing 737 MAX.

## Tecnologías Utilizadas
- HTML5 Semántico.
- CSS3 (Flexbox, Grid y Media Queries).
- Google Fonts (Tipografía profesional).
- Formspree para el formulario de contacto.
